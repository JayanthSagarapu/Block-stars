#Project : 
    Coin Price

#Descripton:
    here we will get the price of a bitcoin for every given interval and then updates it in mongoDb and gets the price from the mongoDb. Sends that response to the frontend.

#Project Setup:
    Install Node.js and npm

    Clone Repository : "https://github.com/JayanthSagarapu/Block-stars.git"

#Install Dependencies : 
        npm Install     - so that all the Dependencies in package.json will be installed
    
    
#Run the application:
        npm start


#To test the application:
        Open "http://localhost:3000" in the browser after running the application
